### Enhancement 2: Entra ID Auth UX

**UI Options Analyzed:**
- **Option 1:** Auth status indicator in webview top bar (proactive feedback)
- **Option 2:** "Sign in with Azure" button (on-demand auth with VS Code OAuth)
- **Option 3:** Trigger `az login` from extension (works with DefaultAzureCredential)
- **Option 4:** Detect auth on first message send (status quo + better error)

**Recommended Hybrid (Options 1 + 2):**
- **Phase 1 (MVP+1):** Auth status indicator with credential source display

- Credential source detection via `DefaultAzureCredential` + VS Code auth fallback

**VS Code Authentication API:**
- ✅ Supports Entra ID with scope: `https://cognitiveservices.azure.com/.default`
- `vscode.authentication.getSession("microsoft", [...], { createIfNone: true })`
- Returns valid bearer token for Azure AI Foundry
