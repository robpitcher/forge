- Event handling uses `.on(eventName, handler)` named pattern
- Send API is `session.send({ prompt })`
- bearerToken is static string only
- Mock patterns for `DefaultAzureCredential` and `SecretStorage`
- "Never throws" functions must catch ALL async operations
- API keys only via SecretStorage
- Air-gap compliance — no external network calls
